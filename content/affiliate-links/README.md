# Affiliate Links

This folder stores public-safe affiliate and partner offer URL references.

Because this repository is public, only store links that are safe to expose publicly.

## Safe to store

- Public affiliate links intended for customer-facing pages.
- Public partner landing pages.
- Public campaign URLs.
- Non-sensitive tracking IDs that are already intended to appear in public URLs.

## Do not store

- API keys.
- Access tokens.
- Passwords.
- Signed URLs.
- Private dashboard URLs.
- Customer data.
- Partner credentials.
- URLs with secret query parameters.

## Recommended files

- `affiliate-links.md` for human-readable notes.
- `affiliate-links.csv` for structured link records.

## Review rule

Before adding a link, confirm it is intended to be public-facing.
